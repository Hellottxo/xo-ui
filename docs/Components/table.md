## API
|参数|说明|类型|默认值|
|---|---|---|---|
|columns|列数组|`object[]`|-|
|dataSource|表格数据|`object[]`|-|
|bordered|是否展示边框|`boolean`|false|
|showHeader|是否展示表头|`boolean`|`false`|
|rowKey|行数据的key，必传|`string`|-|
|maxHeight|最大高度，超出则出现滚动条，单位为px|`number`|-|
|stripe|斑马纹表格|`boolean`|false|
